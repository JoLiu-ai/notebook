# 图标文件说明

请在此目录下放置以下尺寸的图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

## 快速生成图标

你可以使用以下方法生成图标：

### 方法一：使用在线工具
1. 访问 https://www.favicon-generator.org/ 或类似工具
2. 上传一个 512x512 的图片
3. 下载生成的图标文件

### 方法二：使用图片编辑软件
1. 准备一个 512x512 的图标设计
2. 使用 Photoshop、GIMP 等工具导出为不同尺寸
3. 保存为 PNG 格式

### 方法三：使用简单占位图标
如果暂时没有图标，可以使用纯色图片作为占位符：
- 使用任何图片编辑工具创建纯色方块
- 建议使用主题色（如紫色 #667eea）

## 临时解决方案

如果暂时没有图标，插件仍然可以正常工作，只是会显示默认图标。
